doctor-house-api
===

[![Build Status](https://travis-ci.org/holysleeper/doctor-house-api.svg?branch=master)](https://travis-ci.org/holysleeper/doctor-house-api)

API consumed by Dr. House mobile app for:
* fetching categories
* fetching product info (3D model files, textures and so on)
* handling statistics (product hits, last hit, client-product mapping)

Current milestones:
* [done] unit and functional tests + integration with Travis-CI
* [done] Swagger documentation
* [todo] ELK stack integration
